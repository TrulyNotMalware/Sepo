준비물

	1. 라즈베리파이 4 1대 + 3B 1대
	2. 아두이노세트 (세트1개, 보드3개)
	3. 태블릿 (남는거)
	
미세먼지 행동지침 준수 IoT 홈서비스

기능 : 미세먼지 높을날 건강생활수칙을 준수할 수 있게끔 아두이노 등의 IoT기기들을 이용해 사용자에게 알림을 보내주는 서비스
	
	1. 장시간 실외활동 자제
		준비물 : 휴대폰, 휴대폰의 GPS / Cell/WiFi 정보
		작동방식 : 30분마다 위치와 데이터 상태를 체크해 30분동안 이동했고, Celluar 상태이면 실외라고 판단해 알림을 보내줌
			-> 장시간 실외활동을 자제하며, 실내로 이동하라는 권고
			
	2. 외출 시 식약처에서 인증한 보건용마스크 착용
		준비물 : 초음파센서, 태블릿
		작동방식 : 집안에서 바깥으로 나갈 때 초음파센서 (현관문)로 탐지하고, 현재 날씨에 따라 마스크 착용 / 우산 챙기기 등의 권고사항을 태블릿으로 보여줌
		
	3. 외출 후 손, 얼굴 깨끗이 씻기
		준비물 : 초음파센서, 부저
		작동방식 : 바깥에서 집안으로 들어올 때 초음파센서 (현관문)로 탐지하고, 주어진 시간 (30초)내에 화장실로 가지 않으면 (화장실에 있는 초음파센서에 탐지되지 않으면) 부저가 울린다.
		
	4. 충분한 수분섭취
		준비물 : 휴대폰
		작동방식 : 1시간에 한번씩 물마시라고 알림 보내주기
			+ 수분센서를 이용해 텀블러의 물 양을 실시간으로 체크, 얼마나 섭취했는지 확인
		
	5. 창문을 닫아 외부의 미세먼지 유입을 차단
		준비물 : 초음파센서, 휴대폰
		작동방식 : 미세먼지 농도가 높고, 창문이 열려있으면 (초음파센서 탐지) 창문을 닫으라고 권고하고, 미세먼지 농도가 낮고 일조량이 높은 경우에는 창문을 열라는 등의 별도 메시지를 띄워줌
			+ 스티로폼 가 창문같은걸 만들어 자동으로 문이 닫히게끔 구현
			
			

구현하기 위한 메인 서버 필요 (라즈베리파이 4)
 -> 서비스들의 상태나, 미세먼지 농도 등등을 보여주는 대시보드를 띄워줌
